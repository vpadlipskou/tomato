# Tomato Timer

A pomodoro-style timer for getting things done without overthinking it.

Set your time with the wheel, click once, and just flow. When a session ends, you get a sound and move to the chill phase.

## Tomato Timer gives you:

- A visual circular timer on your desktop
- Super simple controls, no button overload
- WORK and REST cycles
- Sound cues so you stay in rhythm

## Tech Stack

- Python
- PySide6

## How to Use

1. Scroll to set your time.
2. Click once to start.
3. Click while running to pause, click again to continue.
4. When time is up, you hear a sound and it switches to the next stage.
5. Need a hard reset? Hold left click for 0.8s.
